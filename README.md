# Online Pokedex
Simple online pokedex that uses API from https://pokeapi.co/

## How to run 
### local machine
```bash
npm install
npm start
# web app will be deployed to http://localhost:3000/edtech-test-pokeapi/ 
# don't forget the baseUrl~ 
```

visit https://dzarrr.github.io/edtech-test-pokeapi/
